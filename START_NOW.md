## âš¡ IF SOMETHING GOES WRONG

### MySQL Not Starting
- Port 3306 might be in use
- Close other database applications
- Restart XAMPP completely
- Check: `C:\xampp\mysql\data\` folder exists

### Apache Not Starting
- Port 80 might be in use
- Close IIS or other web servers
- Check: `C:\xampp\apache\conf\httpd.conf`
- Look at: `C:\xampp\apache\logs\error.log`

### Website Shows Blank
- Clear browser cache (Ctrl+Shift+Delete)
- Check MySQL is running (green light)
- Check Apache is running (green light)
- Visit http://localhost/Royal-Signature/status.php

### Database Connection Error
- Verify MySQL is running
- Verify database exists: `royal_signature`
- Check credentials in `backend/config.php`

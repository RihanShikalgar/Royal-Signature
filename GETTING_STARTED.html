‚ùå "Port already in use" error?

Use a different port. Edit RUN_PHP_SERVER.bat and change 8888 to 8889 or 9000

‚ùå "Cannot find database" error?

The database should already exist. If it doesn't, run the setup file:

‚ùå Login not working?
